# Hotel Management System

A full-stack application for managing hotel operations, including room booking, guest management, and billing.

## Tech Stack
* **Frontend:** React.js, Vite
* **Backend:** Java (Spring Boot)
* **Database:** PostgreSQL

## How to Run
1.  **Backend:** Navigate to `hotel-app` and run the Spring Boot application.
2.  **Frontend:** Navigate to `hotel-frontend`, run `npm install`, then `npm run dev`.
